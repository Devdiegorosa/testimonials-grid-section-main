# 💬 Testimonials Grid Section

A responsive testimonials grid layout built with HTML and CSS.

This project is based on a Frontend Mentor challenge and focuses on creating a responsive grid layout displaying user testimonials.

🔗 Live Demo  
https://devdiegorosa.github.io/testimonials-grid-section-main/

---

## 👨‍💻 About the Project

This project displays a **grid of testimonial cards** where users share their experiences.

The goal of this challenge was to recreate the design as close as possible and practice building responsive layouts using modern CSS techniques.

---

## 🚀 Features

- Responsive layout
- Grid-based design
- Testimonial cards with user information
- Mobile and desktop layouts
- Clean and modern UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- CSS Grid

---

## 🎯 What I Learned

While building this project I practiced:

- Creating layouts using **CSS Grid**
- Building responsive components
- Structuring testimonial cards
- Improving UI design and spacing
